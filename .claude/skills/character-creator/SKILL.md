# Character Creator - 캐릭터 생성 스킬

## Trigger
- "캐릭터 생성", "인물 만들기", "캐릭터 만들기"

## Workflow

### Phase 1: Template Loading
1. `11-템플릿/캐릭터템플릿.md` 로드
2. 기존 캐릭터 목록 확인 (중복 방지)
3. 인물 유형 결정 (헌터/관료/길드원/상인/연구원/민간인)

### Phase 2: Interview
1. **기본 정보**: 이름, 나이, 성별, 역할
2. **외모**: 키/체형, 특징, 장비, 복장
3. **배경**: 출신, 소속 길드/기관, 등급, 파티 역할
4. **성격**: 핵심 특성 3개, 강점, 약점, 두려움
5. **욕구**: 의식적 목표(Want), 무의식적 필요(Need)
6. **능력/전투력**: 생체모방 능력, 전투 스타일, 장비 등급
7. **대화 스타일**: 어휘, 말투, 고유 표현

### Phase 3: Relationship Setting
1. 주인공(태봉)과의 관계 정의
2. 기존 캐릭터와의 관계
3. `03-플롯/인물관계도.md` 업데이트

### Phase 4: Bug Hero Specialization
1. 소속 길드/기관 (및 변화 이력)
2. 등급/파티 역할
3. 태봉과의 합류 계기
4. 주인공에 의해 변화될 관계/입장

### Phase 5: File Creation
1. `02-캐릭터/[역할]/[이름].md` 생성
2. `02-캐릭터/[역할]/[이름]_감정추적.md` 초기화
3. `01-세계관/고유명사사전/인물.md` 등록
