# Bug Hero - 스킬 허브

Bug Hero 프로젝트의 모든 스킬을 관리하는 루트 파일.

## 프로젝트 정보
- **장르**: 현대 타워 판타지 / 먹치킨 / 라이트 코미디
- **주인공**: 구태봉 (해충방제업자, 생체모방 능력)
- **구조**: 화 단위 연재

## 스킬 목록

| 스킬 | 용도 |
|------|------|
| `auto-agents` | 화 집필/수정 파이프라인 오케스트레이션 |
| `scene-writer` | 씬 작성 워크플로우 |
| `character-creator` | 캐릭터 생성 |
| `consistency-check` | 일관성 검사 |
| `entertainment-review` | 3인 페르소나 재미평가 |
| `humanization-review` | AI 흔적 검출 + 인간화 점수 |
| `emotion-analysis` | 캐릭터 감정 추적 |
| `rule-check` | core-rules 기반 퇴고 |
| `prose-polish` | 문단 단위 산문 다듬기 |
| `parallel-refactor` | 5화 병렬 리팩토링 |
| `sync` | Git 브랜치 동기화 |
