# Revision — 통합 퇴고 스킬

초고 또는 수정본에 대해 규칙 검증 → 산문 다듬기 → AI 흔적 제거 → 일관성 검증을 **하나의 파이프라인**으로 실행한다. 개별 퇴고 스킬(rule-check, prose-polish, humanization-review, consistency-check)의 핵심 로직을 통합한 것이며, 실행 강도를 3단계로 선택 가능하다.

## Trigger
- "퇴고", "수정", "revision", "리뷰", "검수", "1차수정", "퇴고 해줘", "다듬기"
- 초고 작성 완료 후 자동 실행 (CLAUDE.md Golden Rules)
- "ep-XXX 퇴고해줘", "이 화 수정해줘"

## 강도 선택

사용자에게 확인하거나, 상황에 맞게 자동 판단:

| 강도 | 실행 단계 | 용도 |
|------|---------|------|
| **신속** | Step 1~3 | 일반 화. 규칙+산문+인간화 |
| **심층** | Step 1~4 | 중요 화. 위에 + 일관성 검증 |
| **최종** | Step 1~5 | 소아크 마무리. 위에 + 재미 평가 |

미지정 시 **신속** 기본. 인간화 점수 70 미만이면 자동으로 심층 전환.

## 입력
- 대상 화 파일 경로 (예: `04-원고/Part1/ep-001_v1_초고.md`)
- 강도 선택 (신속/심층/최종)
- 특정 단계만 실행 가능 (예: "Step 3만 해줘" = humanization만)

## Step 1: 규칙 검증 (from rule-check)

core-rules.md 전 규칙 검사. 위반 발견 시 **즉시 수정**.

### 1A. Grep 기계적 스캔 (즉시 수정)

| 스캔 | Grep 패턴 | 대응 |
|------|----------|------|
| 톤 파괴어 | `스킬트리\|빌드\|너프\|버프\|쿨타임\|어그로\|딜러\|탱커\|힐러\|파밍\|DPS\|PVP\|메타` | 세계관 표현으로 즉시 대체 |
| 수치 시간 | `0\.\d+초\|\d+초간` | 체감 표현으로 교체 |
| em-dash 과다 | `—` (5회 초과 카운트) | 초과분 대체 수단 교체 |
| 분리호칭 | `"\S+\." .+이\|가 말했다\. "` | 합치기/행동 비트 |
| "말했다" 단독 | `" .+이\|가 말했다\.` | 다른 동사/행동 비트 |

### 1B. 서사 구조 검증 (읽기 기반)

| 규칙 | 검사 내용 |
|------|---------|
| §3-1 도입 서사 | 씬 도입이 서술자 선언형인지 확인. 감각→행동→인물 순서인지 |
| §3-2 트리거 무게감 | 복선/트리거 이벤트가 감각적 비트 없이 스윽 지나가는지 |
| §4 문장 분절 | 합칠 수 있는 문장이 분절되어 있는지 (최우선) |
| §4-1 논문체 묘사 | 장소/상황이 보고서식 객관 나열인지. POV 감각 경유인지 |
| §4-2 대화 어순 | 대사에서 SVO 도치가 있는지. 소리 내어 읽었을 때 자연스러운지 |
| §4-3 캐릭터성 | 대사에서 캐릭터 고유 성격이 드러나는지 |
| §5 대화 패턴 | 분리호칭, 동일화자 대사분절, "말했다" 단독 |
| §7 서술 원칙 | Show Don't Tell, 감각 3개+, 대화 노이즈 3회+ |
| §9 대화 진정성 | 정보전달형/감정라벨링/서술형 대사, 리액션 중복, 에코 대사 |
| §10 갈등 깊이 | 비합리성 쿼터 (화당 1회), 갈등 지속 최소 |
| §11 에피소드 종결 | 직전 화와 동일 방식 금지, 5종 순환 |
| §19 선행맥락 | 맥락 없는 대사/행동/사건참조 |
| §20 정보 경계 | 캐릭터가 모르는 정보를 아는 것처럼 |
| §21 어체 일관성 | 존대/반말 혼용, 호칭 불일치 |
| §22 능력 성장 | 3단계(한계→수련→확장) 누락 여부 |

### 1C. 전투씬 포함 시 추가 검증

| 규칙 | 검사 내용 |
|------|---------|
| §0 먹치킨 전투 | 주인공 압도적 우위 유지, 능력 쇼케이스 구조 |
| §14 전투 비중 | 먹치킨 압도 70% / 창의 20% / 고전 10% |
| §23 전투 종결 | 능력 상성→생물학적 해법→압도적 해결 |

### 1D. 분량 검증

화 파일의 글자수를 측정한다 (공백 포함, 개행 제외).

```python
with open(filepath, 'r', encoding='utf-8') as f:
    text = f.read()
count = len(text.replace('\n','').replace('\r',''))
```

| 항목 | 기준 | 미달 시 |
|------|------|--------|
| 화 전체 | 6,000~7,000자 | 가장 짧은 씬을 확장. 내면묘사/감각/대화 비트 보강 |
| 씬 1개 | 2,000~3,500자 | 해당 씬 확장 |

분량 미달은 Step 1 수정 후, Step 2 전에 보강한다. 보강 시에도 core-rules 전 규칙 준수하며, 밀도 있는 확장만 허용 (불필요한 설명/반복/수식어 나열 금지).

### 1E. 수정 적용
위반 발견 시 즉시 Edit으로 수정. 분량 미달 시 확장. 수정 건수를 카운트하며 기록.

## Step 2: 산문 다듬기 (from prose-polish)

Step 1에서 잡히지 않은 문단 단위 미세 문제를 정리한다.

### 검사 항목 (우선순위 순)
1. **어미 반복**: 같은 어미 3문장+ 연속 → 어미 변형
2. **SVO 단문 연속**: 짧은 단문 3문장+ → 접속어미로 통합
3. **정보 쪼개기**: 합칠 수 있는 정보 분절 → 합침
4. **주어 중복 재소개**: 바로 다음 문장에서 재소개 → 생략/합침
5. **시나리오식 컷 나열**: 한 줄 독립 단락 연속 → 흐름 통합
6. **체언 종결 컷**: "타워 3층. 오후." → 접속어미로 합침
7. **단어/표현 중복**: 인접 문장 동일 단어 반복 → 유의어 대체

### 수정 원칙
- 합침 우선. 접속어미 적극 활용
- 합친 결과가 40자+ 되면 다른 분할점 찾기
- 의미/뉘앙스 변경 금지. 위반 수정만 수행
- 의도적 단문(극적 강조 1회)은 보존

## Step 3: 인간화 검증 (from humanization-review)

AI 흔적을 검출하고 인간화 점수를 산출한다.

### 3A. 기계적 패턴 재확인
Step 1에서 이미 수정했지만 잔존분 확인:
- 톤 파괴어, 수치 시간, em-dash, 분리호칭, "말했다" 누적
- SVO 단문 연쇄 (20자 이하 3문장+ 연속)
- 반복 표현 (`입꼬리`, `눈빛.{0,5}흔들`, `어깨.{0,5}축`)

### 3B. 심층 분석
- **대화 자연스러움**: 정보전달 비율, 노이즈 존재, 음성 구분도
- **갈등 해결 속도**: 제기-해결 간격, 잔여 긴장
- **구조 다양성**: 화 성격 유형, 종결 패턴 비연속, 비합리적 행동
- **감각 다양성**: 씬당 감각 종류, 동일 조합 연속, 환경 묘사 변주

### 3C. 점수 산출 (100점 만점)

| 항목 | 만점 | 핵심 기준 |
|------|------|----------|
| 문체 다양성 | 20 | 반복 <3회, 어미 분산, 종결 다양 |
| 대화 자연스러움 | 20 | 노이즈 >20%, 정보전달 <10%, 음성 구분 |
| 구조 비정형성 | 20 | 5종 구조, 종결 비연속, 성장 비균일 |
| 감정 깊이 | 20 | 갈등 3화+, 비합리 행동, 잔여 긴장 |
| 감각 다양성 | 20 | 5종 감각, 조합 비연속, 환경 변주 |

톤 파괴어 1건당 -2점 (최대 -20점)

### 3D. 판정
- **70점 이상**: PASS → Step 4로 (심층 모드 시)
- **70점 미만**: FAIL → `humanization-rules.md` 풀 Read 후 재수정 → 재채점
- 재채점 후에도 미달이면 사용자에게 보고하고 판단 요청

## Step 4: 일관성 검증 (심층/최종 모드만)

작품 전체 문맥에서의 일관성을 검증한다.

### 검증 항목
- **타임라인**: 시간 순서, 이동 시간, 환경 변화
- **캐릭터**: 위치 연속, 정보 범위, 감정 연속, 능력 보유 여부
- **타워/설정**: 층수 규칙, 안전지대 규칙, 등급 일치
- **능력 시스템**: 사용 능력이 해당 시점에 획득 완료인지, 합성 전제 능력 보유 여부
- **복선**: 활성 복선 상태, 30화+ 미회수 경고

### 참조 파일
- `01-세계관/타임라인.md`
- `reference/ability-upgrade-current.md`
- `reference/world-building.md` (타워/능력 검증 필요 시)
- `03-플롯/복선추적.md`

## Step 5: 재미 평가 (최종 모드만)

`entertainment-review` 스킬을 호출하여 3인 페르소나 교차 리뷰 실행.

- 일반독자 (20-30세): 몰입도, 스킵 충동, 감정 이입
- 장르독자 (20-35세): 먹치킨 쾌감, 타워 신선도, 능력 만족도
- 편집자: 페이싱, 패턴 반복, 서사 경제성

100점 만점 + 핵심 약점 TOP 5 + 해결안.
