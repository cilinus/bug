# Prose Polish — 문단 단위 산문 다듬기

## 개요
사용자가 제시한 문단(들)을 core-rules 기준으로 진단하고, 위반 항목을 수정하여 파일에 직접 반영한다.
rule-check(화 전체 감사)과 달리, **지정 문단만 집중 수정**하는 마이크로 퇴고 도구이다.

## 트리거
- "문장 수정", "문단 다듬기", "산문 수정", "룰에 따라 수정", "문장 정리"
- "코어룰 수정", "§4 수정", "문장 단편화 수정"
- 사용자가 문단을 직접 붙여넣고 수정을 요청할 때
- "서술이 지저분함", "문장이 끊겨", "어미 반복" 등 산문 품질 불만 표현 시

## 워크플로우

### Phase 1: 입력 파악
1. 사용자가 문단을 직접 붙여넣었는지, 파일+위치를 지정했는지 확인
2. 파일 지정 시: 해당 파일에서 대상 문단을 Read로 확인
3. **core-rules.md는 이미 자동 주입되므로 별도 Read 불필요** — 진단 기준은 아래 체크리스트 사용

### Phase 2: 문단별 진단
각 문단에 대해 아래 체크리스트를 순서대로 적용한다. 위반이 없는 항목은 건너뛴다.

#### 검사 항목 (우선순위 순)

**A. 어미 반복 (§4)**
- 같은 어미(~했다/~이었다/~있었다) 3문장 이상 연속 → 위반
- 진단 시 실제 어미를 나열하여 보여줄 것

**B. SVO 단문 연속 (§4)**
- 짧은 단문 3문장 이상 연속 → 접속어미(~며/~고/~면서/~한 뒤)로 통합
- 극적 강조 의도 단문 1회는 허용, 연속 사용은 금지

**C. 정보 쪼개기 (§4)**
- 한 문장으로 합칠 수 있는 동작+대상+속성을 여러 문장으로 분절 → 합침
- 전형적 패턴: "X였다. Y였다." (X와 Y가 같은 대상의 속성)

**D. 주어 중복 재소개 (§4)**
- 한 문장에서 소개한 주어를 바로 다음 문장에서 다시 소개 → 합치거나 생략

**E. 시나리오식 컷 나열 (§4)**
- 한 줄짜리 짧은 문장을 독립 단락으로 연속 배치 → 단락 내 흐름으로 통합

**F. 체언 종결 컷**
- "타워 3층. 오후 세 시." 같은 체언 종결 연속 → 접속어미로 합침

**G. 불명확 지시어**
- "그것이", "그것을" 등 지시 대상이 모호한 경우 → 구체화하거나 삭제

**H. 단어/표현 중복**
- 같은 단어가 인접 문장에서 반복 (예: "온기~온기", "다리~다리~다리") → 유의어 대체 또는 합침

**I. 사실 오류**
- 작중 설정과 맞지 않는 서술 → 수정

**J. 기타 core-rules 위반**
- §1 톤 파괴어, §5 대화 패턴, §6 em-dash 상한, §9 대화 진정성, §21 존비어 등
- 해당 위반이 보이면 함께 수정

### Phase 3: 수정 적용
1. **진단 요약을 먼저 제시** — 각 문단의 위반 항목을 규칙 번호와 함께 표로 보여줌
2. **사용자 승인 불필요** — 바로 Edit 도구로 파일 수정 (사용자가 문단을 지정한 시점에서 수정 의사 확인됨)
3. 문단이 여러 개일 경우 순차적으로 Edit 적용

### Phase 4: 변경 보고
각 문단별로 아래 형식으로 보고:

```
**N행:**
- [위반항목]: 구체적 설명
- [위반항목]: 구체적 설명
→ X문장 → Y문장, 어미 Z종 분산
```

## 수정 원칙

### 합침 우선
- 쪼개진 정보를 합치는 것이 1순위. 접속어미(~며/~고/~면서/~한 뒤/~이며/~였으나)를 적극 사용.
- 합친 결과가 지나치게 길어지면(40자+) 다른 분할점을 찾을 것.

### 어미 변형
- 반복 어미를 해소할 때 사용할 수 있는 어미 레퍼토리:
  - 종결: ~했다/~였다/~이다/~않았다/~인 셈이었다/~뿐이었다
  - 접속: ~했으며/~였고/~한 뒤/~면서/~하되/~였으나
  - 전환: 현재형(일반 진리), 추측형(~리라/~을까), 인용형(~이라고/~라 했다)

### 의미 보존
- 원문의 의미, 뉘앙스, 서사적 의도를 바꾸지 않는다.
- 극적 강조를 위한 의도적 단문(1회)은 보존한다.
- 인물 내면 독백의 파편적 흐름은 적절히 유지하되, 규칙 위반은 수정한다.

### 과수정 금지
- 위반이 없는 문장은 건드리지 않는다.
- "더 좋게" 만들기 위한 임의 수정 금지 — 규칙 위반 수정만 수행.

## 예시

### 입력
```
타워 3층까지는 한 시간 거리였다. 서두르면 점심 전에 닿을 수 있었다.
태봉과 단둘이 가는 것이 맞았다. 지난번처럼. 사람을 많이 데려가면 몬스터가 더 몰려왔다.
약점을 보여주는 것이 아니라 약점을 찔러야 잡을 수 있는 놈들이었다.
```

### 진단
- §4 어미 반복: `거리였다/있었다/맞았다/몰려왔다/있었다`
- §4 체언 컷: "지난번처럼."
- §4 정보 쪼개기: 한 시간 거리 + 점심 전 도착 분리
- 단어 중복: "약점을~약점을"

### 수정 결과
```
타워 3층까지는 한 시간, 서두르면 점심 전에 닿을 수 있었다.
지난번처럼 태봉과 단둘이 가는 것이 맞았다. 사람을 많이 데려가면 몬스터가 더 몰려온다.
약점을 보여주는 것이 아니라 찔러야 잡을 수 있는 놈들이었다.
```
→ 6문장 → 4문장, 어미 4종 분산, 체언 컷 해소, 중복 제거
