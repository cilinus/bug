# Party Members - 파티원 프로필 & 아크 설계

주인공 구태봉의 파티원 3명의 프로필, 역할, 성장 아크를 관리한다.

---

## 설계 원칙

### 파티원의 서사적 기능
1. **능력 보완**: 주인공의 약점(페널티, 비전투 시간)을 메우는 역할 분담
2. **코미디 엔진**: 캐릭터 간 케미에서 유머 자연 발생
3. **감정 앵커**: 먹치킨 쾌감 사이의 인간적 온기
4. **성장 거울**: 각자의 트라우마 극복이 주인공의 성장과 공명

### 규칙
- 파티원은 Part 1 (1~30화) 안에 전원 합류 완료
- 합류 순서: 소라(5~8화) → 해민(12~15화) → 기택(20~25화)
- 각 파티원의 내면 아크는 **최소 3개 파트에 걸쳐** 전개
- 파티원 POV 씬은 전체의 20% 이내 (주인공 POV 80%+)

## 1. 한소라 (韓素羅, 28세, 여) — 탱커/서포터

| 항목 | 내용 |
|------|------|
| 이름 | 한소라 |
| 나이 | 28세 |
| 전직 | 소방관 (3년차). 건물 붕괴 사고로 동료 3명 사망 시 각성 |
| 각성 능력 | 강화 체질 — 체력·방어력·회복력 극대화. C급 |
| 성격 | 정의감 강함, 직선적, 솔직. 남 돌보는 것에 익숙하지만 본인 감정은 잘 말하지 않음 |
| 파티 역할 | 방패(전면 탱킹) + 상식인. "그게 진짜 되는 거야?" 전담 |
| 시그니처 | 태봉의 능력 부작용 볼 때마다 기겁. "아 진짜 소름…!" |

### 합류 계기 (5~8화)
- 타워 저층에서 고립 상태. 강화 체질이지만 단독 공략 한계
- 태봉이 기절염소(擬死) 능력으로 몬스터 주의를 분산, 소라를 방패 삼아 돌파
- "저를… 방패로 쓴 거예요?" / "강화 체질이잖아요. 가장 합리적인 선택이죠"
- 어이없지만 결과적으로 살아남은 경험 → 파티 결성

### 내면 아크
1. **동료 상실의 트라우마**: 소방관 시절 동료 3명을 구하지 못한 죄책감
2. **"이번엔 지킨다"**: 파티원을 과잉보호. 태봉의 위험한 행동에 매번 제동
3. **전환점**: 태봉이 "지키려고만 하면 같이 싸울 수 없어요"라고 직격
4. **완성**: "지키는 것이 아니라 함께 싸우는 것" — 방패가 아니라 동료로

### 대화 스타일
- 초반 해요체 → 친해지면서 반말 전환 (Part 1 후반)
- 직설적, 감정 표현 솔직. 돌려 말하지 않음
- 화나면 말이 짧아짐. "됐어." "나가." "닥쳐."
- 태봉에게: "오빠" (한 살 차이지만 자연스럽게 정착)

### 전투 스타일
- 전면에서 막기. 몬스터 어그로 집중
- 태봉이 능력 사용할 시간 벌어주기
- 가끔 물리력으로 몬스터를 직접 날려버림 (파워 탱커)
- 소방관 시절 구조 기술 활용 (밧줄, 진입, 인명구조)

### 감정 추적 기준 (8단계)

| 단계 | 감정 상태 | 트리거 |
|------|----------|--------|
| E1 | 경계/불신 | 첫 만남, 태봉의 비상식적 행동 |
| E2 | 어이없음 + 약간의 인정 | 태봉의 능력이 실제로 먹힘 |
| E3 | 동료 의식 형성 | 함께 위기를 넘김 |
| E4 | 과잉보호 모드 | 태봉의 부작용에 불안 |
| E5 | 갈등 (보호 vs 신뢰) | 태봉의 직격 발언 |
| E6 | 신뢰로 전환 | 믿고 맡기는 첫 순간 |
| E7 | 진정한 파트너십 | 함께 싸우는 것의 의미 체득 |
| E8 | 완성된 동료 | 트라우마 극복, 대등한 관계 |

### 코미디 포인트
- 태봉의 벌레 능력에 대한 리액션 (벌레 싫어함)
- 강화 체질이라 아픔을 잘 못 느낌 → 부상 자각 늦음
- 소방관 습관: 위험 상황에서 자동으로 "대피하세요!" 외침
- 해민의 요리에 감동받아 눈물 (전투 중에는 무표정인데)

### 성장 마일스톤

| 시점 | 마일스톤 | 연관 파트 |
|------|---------|----------|
| Part 1 | 파티 합류, 태봉 능력에 적응 | 합류~안정 |
| Part 2 전반 | 과잉보호 갈등 정점 | 갈등 아크 |
| Part 2 후반 | "함께 싸운다"로 전환 | 성장 전환 |
| Part 3 | 강화 체질 각성 심화, B급 돌파 | 능력 성장 |
| Part 4 | 트라우마 완전 극복, 독립 행동 가능 | 완성 |

## 2. 정해민 (鄭海旻, 26세, 남) — 딜러

| 항목 | 내용 |
|------|------|
| 이름 | 정해민 |
| 나이 | 26세 |
| 전직 | 호텔 셰프 지망생. 각성 시 주방 화재, 호텔 전소, 해고 |
| 각성 능력 | 화염 조작 — F~D급 중 가장 흔한 능력. D급 |
| 성격 | 수다쟁이, 낙천적, 먹는 것에 진심. 겉은 가벼워 보이지만 자기비하 깊음 |
| 파티 역할 | 화력 + 밥 담당. 태봉의 대사 과부하 페널티 전담 해결사 |
| 시그니처 | 전투 후 몬스터 식재료로 요리. "형, 이거 구우면 의외로 맛있어요" |

### 합류 계기 (12~15화)
- 안전지대에서 태봉이 대사 과부하 페널티로 배고파 쓰러져 있음
- 해민이 몬스터 식재료로 즉석 요리를 해줌
- "형씨, 나랑 파티 하면 밥값은 확실히 벌어"
- 태봉: "계약서 어디 있어요?" (즉답)

### 내면 아크
1. **파괴자 트라우마**: 각성의 불이 호텔을 태움. "나는 태우기만 하는 놈"
2. **요리로 증명**: 불을 파괴가 아니라 창조에 쓰려는 시도
3. **전환점**: 파티원을 위한 요리가 전투력 버프로 작용하는 순간
4. **완성**: "불은 태우는 게 아니라 익히는 거야" — 능력과 정체성의 화해

### 대화 스타일
- 반말 기본. 태봉에게 "형" 호칭
- 수다스러움, 음식 비유 자주 사용 ("이건 좀 덜 익었는데요" = 준비 부족)
- 진지한 말을 할 때 갑자기 조용해짐 → 파티원이 놀람
- 소라에게: "누나" (처음엔 "소라 씨" → 자연스럽게 전환)
- 기택에게: "아저씨" → "기택이 형" (기택이 싫어하지만 묵인)

### 전투 스타일
- 원거리 화염 지원. 범위 딜
- 화력 조절 능력 성장 (처음엔 강약 조절 서투름)
- 몬스터 식재료 판별 능력 (셰프 경험 활용)
- 전투 후 즉석 요리 → 파티 HP/스태미나 회복

### 감정 추적 기준 (8단계)

| 단계 | 감정 상태 | 트리거 |
|------|----------|--------|
| E1 | 자기비하 + 겉 밝음 | 평소 상태 |
| E2 | 태봉에 대한 호기심 | 벌레 능력의 비범함 |
| E3 | 소속감 형성 | 파티에서 자기 역할 발견 |
| E4 | 파괴 트라우마 재발 | 화력 폭주 사고 |
| E5 | 자기 부정 극심 | "나는 태우기만 해" |
| E6 | 요리의 가치 재발견 | 요리 버프 효과 입증 |
| E7 | 능력 수용 | 불과 화해 |
| E8 | 완성된 셰프 헌터 | 파괴와 창조의 통합 |

### 코미디 포인트
- 모든 상황을 음식에 비유 ("이 던전 레시피가 좀 별로네")
- 몬스터를 보면 식재료 판별부터 함 ("저거 살코기 많겠다")
- 요리 퀄리티에 자존심 강함. 맛없다고 하면 진짜로 상처받음
- 태봉의 대사 과부하 시 긴급 영양 공급 → "형, 밥이 약이에요!"

### 성장 마일스톤

| 시점 | 마일스톤 | 연관 파트 |
|------|---------|----------|
| Part 1 | 파티 합류, 화력 조절 훈련 | 합류~적응 |
| Part 2 전반 | 화력 폭주 사고, 트라우마 재발 | 갈등 아크 |
| Part 2 후반 | 요리 버프 시스템 확립 | 역할 확립 |
| Part 3 | 화염 조작 정밀화, C급 돌파 | 능력 성장 |
| Part 4 | "불은 익히는 것" 체득, 독자 능력 개발 | 완성 |

## 3. 윤기택 (尹基澤, 45세, 남) — 정보/참모

| 항목 | 내용 |
|------|------|
| 이름 | 윤기택 |
| 나이 | 45세 |
| 전직 | 보험 설계사 (18년차). 타워 출현 후 보험 시장 붕괴, 이혼, 딸(윤서아, 16세)과 별거 |
| 각성 능력 | 확률 직감 — 위험도를 0~100 수치로 감지. E급 (전투 불가) |
| 성격 | 신중, 계산적, 말수 적지만 핵심만 말함. 중년의 쓴맛과 자조 유머 |
| 파티 역할 | 두뇌 + 협상. 길드/정부 계약, 아이템 매매, 수익 관리 |
| 시그니처 | 태봉 옆 위험도 = "0" 또는 "측정 불가". "... 이건 보험 설계가 안 됩니다" |

### 합류 계기 (20~25화)
- 태봉의 첫 길드 마찰 (크라운 길드와 수익 분배 분쟁)
- 기택이 법적/행정적 문제를 깔끔하게 정리
- "공짜입니다. 대신 파티에 넣어주세요"
- 태봉: "보험 안 팔 거죠?" / 기택: "... 누가 들어줍니까"

### 내면 아크
1. **인생 끝났다**: 보험 시장 붕괴, 이혼, 딸과 단절. "나는 끝난 사람"
2. **측정 불가의 변수**: 태봉이라는 예측 불가 존재에 흥미. 계산이 깨지는 쾌감
3. **계산 밖의 인생**: 파티 생활에서 숫자로 환산 못 하는 가치 발견
4. **완성**: 딸에게 연락 → 화해. "확률 밖의 일도 일어나더라"

### 대화 스타일
- 합쇼체 기본. "구 대표" 호칭 (태봉이 대표가 아닌데도)
- 격식체, 핵심만 말함. 불필요한 말 없음
- 자조적 유머: "제 나이에 던전을 뛰다니, 보험료가 천문학적이겠군요"
- 진지할 때 안경 고쳐 씀 (습관)
- 소라에게: "한 씨" / 해민에게: "정 군"

### 전투 스타일
- 비전투원. 직접 교전 불가
- 위험도 측정으로 최적 루트 안내, 트랩 감지, 매복 경고
- 태봉 옆에서 위험도가 "0"이 되거나 "측정 불가"가 뜨는 현상 → 개그 + 복선
- 정보 분석: 몬스터 패턴, 타워 구조, 적 헌터 동향 파악

### 감정 추적 기준 (8단계)

| 단계 | 감정 상태 | 트리거 |
|------|----------|--------|
| E1 | 체념/자조 | 합류 초기 상태 |
| E2 | 직업적 흥미 | 태봉의 수익성(?) 분석 |
| E3 | 동료 의식 발아 | 파티의 비효율적이지만 따뜻한 관계 |
| E4 | 딸 생각 | 소라/해민의 젊은 에너지에 촉발 |
| E5 | 계산 밖의 감정 | 파티를 위해 비합리적 선택 |
| E6 | 딸에게 연락 시도 | 용기 |
| E7 | 화해 과정 | 서투른 소통 |
| E8 | 완성된 가족 | 파티 = 두 번째 가족 |

### 코미디 포인트
- 모든 상황을 보험/리스크로 환산 ("이 던전의 손해율은...")
- 태봉 옆 위험도가 버그처럼 작동 → 기택만 혼란
- 보험 서류 가방을 던전에도 들고 다님 (... 습관)
- 해민이 "아저씨" 부를 때마다 미간 찡그림
- 중년 체력 한계: 계단 많은 층에서 헐떡임

### 성장 마일스톤

| 시점 | 마일스톤 | 연관 파트 |
|------|---------|----------|
| Part 1 | 파티 합류, 길드 협상 능력 입증 | 합류~역할 확립 |
| Part 2 전반 | 딸과의 관계 떠오름, 갈등 | 내면 아크 |
| Part 2 후반 | 파티를 위한 비합리적 선택 | 전환점 |
| Part 3 | 딸에게 연락, 서투른 화해 시작 | 관계 회복 |
| Part 4 | 가족과 화해 완성, 확률직감 응용력 확장 | 완성 |

## 파티 케미 매트릭스

| 관계 | 케미 유형 | 대표 상호작용 |
|------|----------|-------------|
| 태봉 ↔ 소라 | 비상식 vs 상식 | "그게 되는 거야?" / "됐잖아요" |
| 태봉 ↔ 해민 | 먹치킨 콤비 | 능력 + 요리 = 최강 시너지 |
| 태봉 ↔ 기택 | 변수 vs 계산 | 위험도 "측정 불가" 개그 |
| 소라 ↔ 해민 | 누나 + 동생 | 소라가 해민 챙기고 해민이 밥 해줌 |
| 소라 ↔ 기택 | 정의감 vs 현실주의 | 가끔 충돌, 서로 인정 |
| 해민 ↔ 기택 | 수다 vs 과묵 | 해민이 일방적 수다, 기택은 한마디로 정리 |
