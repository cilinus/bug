# Core Rules - Sub-Agent 집필용 핵심 규칙

Phase 3 병렬 집필 Sub-agent에게 자동 주입되는 규칙 요약본.
전체 규칙은 `.claude/reference/` 하위 파일 참조. 이 파일은 집필 시 필수 최소 규칙만 포함한다.

---

## 1. 산문 형식

- 마크다운 형식 절대 금지 (순수 산문만)
- POV: 3인칭 제한 시점, 씬 내 POV 전환 절대 금지
- 시제: 과거형
- 톤: 유쾌한 코미디(50%) + 먹치킨 액션(35%) + 따뜻한 인간 드라마(15%)
- 서술형 완결문 기본. 파편적 단문 나열 금지 (연속 2문장 이상 단문 금지)
- 내면 묘사 풍부하게 — 생물학 지식, 해충방제 비유, 판단 근거를 서사로 채운다
- 능력 해설 상세하게 — 독자가 생물학을 몰라도 이해하도록
- 씬 전환: `* * *`
- 대화: 큰따옴표

## 2. 절대 금지 패턴

### 톤 파괴 패턴
- 과도한 내면 고뇌 (3문장 이상 연속 어둡기) — Bug Hero는 밝은 라이트 코미디
- 비장한 영웅적 독백/연설
- 잔혹한 부상/고어 묘사
- 주인공의 장기 좌절/무기력
- 주인공 말투의 갑작스러운 격식화

### AI 흔적 최상위 패턴
- **SVO 단문 연쇄**: "SVO. SVO. VO." 3문장 이상 연속 절대 금지
  - SF01: 체언 종결 파편("흔들었다. 작게.") → 수식어 통합
  - SF02: 부정-긍정 이중 서술("X 아니었다. Y였다.") → 직접 서술
  - SF03: 동시/연속 동작 분리("열었다. 꺼냈다.") → ~면서/~한 뒤 접속. 단, 대상 문장에 이미 전환(-었다가)이 있으면 통합 금지
  - SF04: 수치적 동작 나열("세 동작. 두 번 만에.") → 인물 특성으로 서술
  - SF05: 설명 꼬리표("그것을 알았다.") → 이미 전달된 정보면 삭제
- **분리 호칭 패턴**: `"기택." 태봉이 말했다. "내용"` → 합치거나 행동 비트로 전환
- **`"[대사]." [이름]이/가 말했다.` 단독 패턴**: 수식어 없이 대사+말했다만 → 태그 삭제, 행동 비트, 또는 다른 동사(답했다/받았다/덧붙였다)
- **수치 시간**: "0.1초", "0.3초간" → 체감 표현("심장이 한 번 뛰는 시간")
- **화 번호 참조**: "X화에서" → 구체적 시간/장소/사건으로 대체
- **메타 서술 라벨**: "능력 해석이 시작됐다", "아이디어가 떠올랐다" → 라벨 없이 직접 침잠

### em-dash(—) 규칙
- 화당 최대 5회, 같은 문단 2회 이상 금지
- 접속사 직후 em-dash 금지, 열거 총괄 금지, 삽입어구 쌍 금지

## 3. 대화 규칙

### 현대 대화 스타일
- 태봉: 해요체 + 해충방제 비유, 담백하고 실용적
- 소라: 직설적 반말, 단호
- 해민: 수다스러운 반말 + 음식 비유, 밝음
- 기택: 격식체 + 데이터 용어, 핵심만

### 필수 대화 노이즈 (화당 최소 3회)
끊김 / 딴소리 / 비완성 문장 / 오해 / 침묵 중 3종 이상

### 대화 비율
대화 ~40% / 행동·묘사 ~35% / 내면 묘사 ~25%

### 금지
- 정보전달형 대사 (상대가 아는 정보를 독자를 위해 설명)
- "말했다" 수식어 없이 단독 사용 (위 참조)

## 4. 감각 묘사

- 매 씬 최소 3개 감각, 시각 편중 금지
- 현대 타워 감각: 타워 조명/몬스터 외형/능력 이펙트/충격파/신체 변화/던전 습기/영양바
- 해충방제 감각 (주인공): 약품 냄새, 어두운 공간 방향감각, 습도/온도 변화

## 5. Show Don't Tell

- 감정 직접 서술 금지 → 신체 반응으로
- 능력 사고 직접 서술 금지 → 생물학 지식 연상으로
- 위기씬: 분석적 거리 금지 → 즉각적 패닉(파편적 사고, 딜레마)

## 6. 분량

- 씬 1개: 2,000~3,500자 목표, 최소 1,500자
- 화 전체(2-3씬): 6,000~7,000자, 최소 5,000자

## 7. 에피소드 종결 방식 (5가지 순환)

1. 끊김형 / 2. 대화 잔향형 / 3. 감각 단독형 / 4. 질문형 / 5. 타인 시선형
→ 아웃라인에 지정된 종결 유형을 따른다. 직전 에피소드와 동일 방식 금지.

## 8. 코미디 & 비합리성

- **코미디 비트**: 씬당 최소 1개 유머 요소 필수
- **페널티 유머**: 생체모방 사용 후 반동(배고픔/외형변화/졸림) 묘사 누락 금지
- **비합리성 쿼터**: 화당 최소 1회 — 비합리적 선택, 감정에 의한 실수, 자기기만, 회피, 과잉반응 중 하나
