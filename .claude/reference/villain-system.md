# Villain System - 악역 시스템

악역 설계 템플릿, 아크별 악역 배정, 응징 체크리스트를 정의한다.
순수 악인 금지. 독자가 "일리 있다"고 느껴야 한다.

---

## 악역 아크 4단계 템플릿

### 1단계: 등장(위협) — 1화
- 악역의 존재와 위협을 독자에게 각인
- 악역의 동기/배경 암시 (순수 악 금지 — "반은 맞는 말" 설계)
- 주인공 측이 위협을 인지

### 2단계: 에스컬레이션(피해) — 1-2화
- 주인공 또는 파티원에게 **직접적 피해** (필수)
- 피해 유형: 등급 심사 방해, 장비 파손, 파티 분열 유도, 타워 내 방해, 사회적 압박
- 악역이 우위에 있는 구간 — 독자 분노 축적

### 3단계: 대결 — 1화
- 주인공 측과의 정면 대결 (전투 또는 비전투)
- 악역의 능력/전략이 충분히 발휘됨
- 먹치킨 톤이지만 악역도 대단해 보여야 함

### 4단계: 응징(카타르시스) — 1화
- 악역의 체면 손상/자멸/겸허한 인정 = E9 쾌감 패턴
- 주인공이 직접 또는 주도적으로 처리
- **응징 수위**: 물리적 파괴 금지. 체면 손상, 자기모순 노출, 실력 차이 인정이 원칙
- 응징 후 잔여 영향 1-2화 유지

---

## "반은 맞는 말" 설계 원칙

모든 빌런은 **그들 나름의 논리**를 가져야 한다. 독자가 "저 사람 말도 일리가 있는데..."라고 느껴야 빌런에게 깊이가 생긴다.

### 핵심 빌런 설계

| 빌런 | 주장 (표면) | 진짜 동기 (이면) | 주인공과의 갈등 축 |
|------|-----------|----------------|------------------|
| 강현우 | "질서와 시스템이 헌터를 지킨다" | 실은 기득권 유지 | 질서 vs 자유 |
| 백서진 | "예외를 허용하면 시스템이 무너진다" | 실은 통제욕 | 시스템 vs 예외 |
| 니콜라이 | "개인의 힘은 집단에 귀속되어야 한다" | 실은 자존심 (최강자는 자기여야) | 집단 vs 개인 |

### 설계 규칙
- 빌런의 주장이 50% 이상 틀리면 안 된다 — 주장 자체는 일리가 있어야 함
- 이면의 동기가 주장을 오염시키는 구조 — "맞는 말을 하는데 속셈이 다르다"
- 주인공이 빌런의 주장 중 맞는 부분은 인정하는 장면 필수 — "그 말 자체는 맞아요"
- 응징은 빌런의 이면(진짜 동기)이 드러나는 순간과 겹쳐야 함

## 악역 설계 요소

### 필수 항목
| 항목 | 설명 |
|------|------|
| 이름/직함 | 네임드 악역 (이름 없는 악역 금지) |
| 동기 | 납득 가능한 적대 이유 + "반은 맞는 말" |
| 능력 | 주인공에게 실질적 위협이 되는 능력 (전투력, 권력, 사회적 영향) |
| 피해 대상 | 누구에게 어떤 피해를 가하는가 |
| 응징 방식 | 어떻게 체면이 손상되는가 |
| 지속 기간 | 최소 3화 |

### 악역 유형
| 유형 | 정의 | 예시 |
|------|------|------|
| 라이벌 헌터 | 능력/실력 경쟁 | 상위 등급 헌터, 경쟁 파티 |
| 체제 수호자 | 제도/시스템을 방패로 방해 | 협회 관료, 길드 간부 |
| 외부 위협 | 타워 외부/타 조직의 적 | 외국 헌터, 불법 조직 |
| 내부 방해자 | 아군 내 갈등/배신 | 파티 내 불화, 길드 내 정치 |

## 응징 체크리스트

악역 아크 완료 시 아래 항목 점검:

- [ ] 악역이 네임드인가? (이름 있는가)
- [ ] 악역의 동기가 납득 가능한가? ("반은 맞는 말"인가)
- [ ] 주인공/파티원에게 직접적 피해를 가했는가?
- [ ] 피해가 구체적인가? (추상적 위협이 아닌가)
- [ ] 악역 지속 기간이 3화 이상인가?
- [ ] 주인공이 주도적으로 응징했는가?
- [ ] 응징 수위가 적절한가? (체면 손상/자멸/인정, 과도한 물리적 파괴 아닌가)
- [ ] 빌런의 이면 동기가 드러났는가?
- [ ] E9 쾌감 패턴으로 기록했는가?
