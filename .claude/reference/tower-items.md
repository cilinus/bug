# Tower Items - 타워 아이템 시스템

Bug Hero의 아이템 체계를 관리한다.
주인공의 "무기"는 몸 자체(생체모방). 아이템은 서포트·보상·장비 중심.

---

## 설계 원칙

### 아이템의 서사적 기능
1. **페널티 경감**: 주인공의 대사 과부하·감각 잔상·역변이를 완화하는 소모품
2. **파티 케미 강화**: 해민의 요리, 기택의 매매, 소라의 장비 개조 등 캐릭터별 아이템 연계
3. **경제 시스템**: 마나석 = 화폐. 길드와의 수익 분쟁, 기택의 협상 씬 기반
4. **코미디 소재**: 해충방제 장비 개조, 보험 서류 가방 등

### 금지
- 먹치킨 파괴 아이템 (주인공 능력을 무효화하는 아이템 금지)
- 아이템 의존 성장 (성장은 능력 자체의 발전으로. 아이템은 보조)
- 한 화에 신규 아이템 3개 이상 등장 (인플레 방지)

| 장비 설계도 | 방호구, 통신기, 특수 도구 제작법 | 안전지대 공방에서 제작 |

|--------|------|------|------|

## 2. 지원 아이템 (소모품)

### 페널티 경감 계열

| 아이템 | 대응 페널티 | 효과 | 비고 |
|--------|-----------|------|------|
| 영양팩 시리즈 | 대사 과부하 | 급격한 칼로리 소모 보충 | 해민 특제 요리가 상위호환 |
| 감각 완화제 | 감각 잔상 | 과민해진 감각 일시 둔화 | 귀마개, 차광 고글, 촉각 억제 장갑 |
| 변이 억제제 | 신체 역변이 | 역변이 지속시간 단축 | 희귀. 고가 |
| 에너지 부스터 | 슬롯 한계 | 동시 사용 슬롯 일시 +1 | 사용 후 반동 큼. 긴급용 |

### 영양팩 vs 해민 요리 비교

| 항목 | 시판 영양팩 | 해민 특제 요리 |
|------|-----------|--------------|
| 칼로리 보충 | 즉각, 단순 보충 | 느리지만 지속적 회복 |
| 맛 | 형편없음 | 맛있음 (사기 진작) |
| 추가 효과 | 없음 | 층별 식재료에 따라 버프 (저항, 회복 등) |
| 비용 | 마나석 소모 | 식재료 + 해민의 노동 |
| 코미디 | 태봉이 맛없다고 투덜 | 해민이 자랑, 태봉이 묵묵히 세 그릇 |

## 3. 장비

### 구태봉 (주인공)
- **기본 원칙**: 특수 장비 거의 안 씀. 몸이 무기
- **해충방제 장비 개조**: 분무기 → 마나석 분사기, 방제복 → 간이 방호복, 끈끈이 트랩 → 몬스터 포획용
- **코미디 포인트**: S급 헌터들이 최첨단 장비를 쓰는데 태봉은 방제 장비. "그거 바퀴벌레 잡는 거 아니에요?" / "몬스터도 해충이잖아요"
- **유일한 고급 장비**: 통신기 (파티 연락용). 기택이 강력 추천해서 구매

### 한소라 (탱커)
| 장비 | 설명 | 특이사항 |
|------|------|---------|
| 강화 방패 | 마나석 강화 합금. 대형 라운드실드 | 소방관 시절 방화문 부수던 감각으로 사용 |
| 방화복 개조 방호구 | 소방용 방화복 기반 경량 방호구 | 내열·내충격. "이건 500도까지 버텨요" |
| 강화 장갑 | 맨손 격투 보조 | 몬스터 직접 잡아 던지기용 |

### 정해민 (딜러)
| 장비 | 설명 | 특이사항 |
|------|------|---------|
| 조리도구 겸 무기 | 프라이팬(근접), 식칼(투척) | "이건 무기가 아니라 도구예요" (본인 주장) |
| 내열 조리복 | 자체 화염에 안 타는 특수 소재 | 셰프복 디자인. 해민의 자존심 |
| 식재료 보관함 | 온도 유지 마나석 내장 보관함 | 전투 중에도 식재료 신선도 유지 |
| 향신료 파우치 | 허리에 차는 소형 파우치 | 요리용이지만 가끔 몬스터에 뿌림 (효과 있음) |

### 윤기택 (정보/참모)
| 장비 | 설명 | 특이사항 |
|------|------|---------|
| 태블릿 | 타워 정보 분석, 몬스터 DB, 시세 확인 | 기택의 생명줄. 배터리 걱정 상시 |
| 보험 서류 가방 | 가죽 서류 가방 | 전직 습관. 안에 분석 자료·계약서·비상금 |
| 위험도 측정 로그북 | 수기 기록장 | 위험도 수치 패턴 분석용. 아날로그 백업 |
| 경량 방호조끼 | 최소한의 방어구 | "저는 맞으면 죽습니다" — 본인의 솔직한 평가 |

## 4. 아이템 경제 규칙

- 마나석은 **파티 공동 관리** (기택이 회계)
- 보스 드롭 처분은 **파티 합의** (기택이 시세 조사 후 제안)
- 길드 수수료, 세금, 장비 유지비 등 현실적 비용 반영
- 아이템 거래 씬 = 기택의 협상 능력 + 코미디 (깎는 기택 vs 안 깎아주는 상인)
- **아이템 인플레 방지**: 새 아이템은 기존 아이템의 상위호환이 아니라 **상황별 선택지** 추가
