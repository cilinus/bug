# Villain POV - 빌런 POV씬 배치표

주인공 외 빌런 시점 씬의 배치와 서술 가이드를 관리한다.

---

## 설계 원칙

### 빌런 POV의 목적
1. **"반은 맞는 말" 제시**: 빌런의 논리가 일리 있음을 보여주되, 독자는 주인공 편을 들게
2. **위협감 구축**: 빌런이 주인공을 어떻게 인식하고 대응하는지 직접 보여줌
3. **세계관 확장**: 기득권 길드, 정부 관료, 해외 헌터의 시각으로 세계 다층화
4. **긴장감 유지**: 주인공이 모르는 위협을 독자만 아는 드라마틱 아이러니

### 규칙
- 전체 비율: **주인공 POV 80% / 빌런+파티원 POV 20%**
- 빌런 POV는 **씬 단위 전환만** (씬 내 전환 금지)
- POV 캐릭터의 감각/사고만 직접 서술 (3인칭 제한 시점 유지)
- 빌런 POV 씬은 해당 빌런의 **서사적 활동 파트에 집중** 배치

## Part별 빌런 POV 배치

### 강현우 POV — Part 1~2 집중 (10화당 1회)

| 화수 범위 | 씬 내용 | 서사적 기능 |
|----------|--------|-----------|
| 8~12화 | 하급 헌터(태봉)의 독점 파괴 보고를 받음 | 기득권자의 첫 인식 — "별것 아닌 놈" |
| 18~22화 | 태봉 파티의 성장세 분석, 첫 견제 지시 | 위협 인식 상승 — "관리가 필요하다" |
| 28~32화 | 견제 실패, 직접 개입 결정 | 분노 + 합리화 — "질서를 지키는 것" |
| 38~42화 | 중력으로 태봉을 압도하려는 계획 | 절대 강자의 자신감 |
| 48~55화 | 패배/좌절 후 재기 or 몰락 | 클라이맥스 — 빌런 아크 종결 |

### 백서진 POV — Part 2 집중 (10화당 1회)

| 화수 범위 | 씬 내용 | 서사적 기능 |
|----------|--------|-----------|
| 30~35화 | 태봉의 능력 보고서 검토, 분류 불가 판정 | 관료의 불안 — "통제할 수 없는 존재" |
| 40~45화 | 태봉 견제 법안/규제 추진 | 합법적 탄압 시도 — "법 안에서 제거" |
| 50~55화 | 규제 실패, 여론 역풍 | 관료적 한계 노출 |
| 60~65화 | 태봉과의 직접 대면/협상 or 몰락 | 클라이맥스 — "통제"의 허상 자각 |

### 니콜라이 볼코프 POV — Part 3 집중 (10화당 1회)

| 화수 범위 | 씬 내용 | 서사적 기능 |
|----------|--------|-----------|
| 60~65화 | "인류의 검"으로서의 자부심, 태봉 정보 입수 | 최강자의 여유 — "흥미로운 벌레잡이" |
| 70~75화 | 태봉의 고층 공략 소식, 경쟁 의식 | 자존심 균열 — "내 자리를 위협?" |
| 80~85화 | 직접 대결 결심 | 절대 강자의 투지 — "증명해야 한다" |
| 90~95화 | 대결 전후 | 클라이맥스 — 패배 or 인정 |

## POV 인물별 서술 가이드

### 강현우 (S급, 중력 조작, 크라운 길드장) POV
- **사고 방식**: 효율주의, 기득권 유지 논리. "약한 헌터가 자원을 낭비하면 인류 전체의 손해"
- **내면 톤**: 자신감 + 지배욕. 평소 차분하지만 통제가 깨지면 분노
- **"반은 맞는 말"**: S급이 자원을 독점하는 것이 효율적이라는 논리는 일면 타당
- **태봉에 대한 인식**: 무시 → 짜증 → 경계 → 위협 → "이해 불가"
- **금지**: 순수악 묘사. 강현우에게도 지키려는 것(길드원, 질서)이 있어야 함

### 백서진 (비각성자, 헌터관리국 부국장) POV
- **사고 방식**: 관료적 완벽주의. 모든 능력은 분류·관리·통제 가능해야 함
- **내면 톤**: 불안 + 사명감. 비각성자로서 각성자 세계를 통제해야 하는 긴장
- **"반은 맞는 말"**: 통제 불가 능력이 사회에 위협이 될 수 있다는 우려는 합리적
- **태봉에 대한 인식**: 분류 불가 → 위협 요소 → 협상 대상 → (선택지: 화해 or 몰락)
- **금지**: 단순 악역 관료. 백서진의 규제에도 나름의 논리와 선의가 있어야 함

### 니콜라이 볼코프 (러시아, SS급 추정, 빙결 지배) POV
- **사고 방식**: 무인(武人)의 자존심. 최강자로서의 책임감과 고독
- **내면 톤**: 고독 + 투지. 정상에 선 자의 외로움. 러시아어 내면 독백 간헐 삽입
- **"반은 맞는 말"**: 인류를 지키려면 최강자가 필요하고, 그것은 자신이어야 한다
- **태봉에 대한 인식**: 무관심 → 호기심 → 라이벌 의식 → 진정한 대결 상대
- **금지**: 러시아 스테레오타입. 입체적 무인으로 그릴 것
