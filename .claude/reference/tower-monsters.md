# 타워 몬스터 설정 — 압축 참조본

> 상세본: `01-세계관/타워몬스터설정/타워몬스터_1-50층.md`, `타워몬스터_51-100층.md`

## 핵심 규칙

- 타워 내부는 **하나의 공유 공간** (입구 8곳, 내부 동일)
- 각국 층수 = **공략 진도** (한국 50, 미국 60, 러시아 70, 메인 타워 100)
- 5층 단위 생태 구간(총 20개), **5층마다 중간보스**, **10층마다 보스**
- 안전지대: 10층마다 (10, 20, 30, 40, 50, 60, 70, 80, 90층)
- 몬스터 = 실존 생물 변이체 기반, 학명·메커니즘 명확
- 능력 각성: 2~3층 클리어당 1회, 1화 신규 2개 초과 금지
- 주인공 처리 방식 = 반드시 **해충방제 지식** 연결

## 100층 최종보스: 오리진 (4페이즈)

1. **환경 극한**: 타워가 중력/온도/기압 조작 → 물곰 내성+불사조 재생으로 버팀
2. **방어층 돌파**: 결정 갑각(공명파쇄) → 균사(기생 역전) → 항체(완전흡수)
3. **코어 도달+대화**: "왜 인간을 시험하지?" / "선별이야" / "나는 다 잡거든요"
4. **최종**: 완전생명체 상태+코어 접촉 → 에너지 역류 → **타워 정지**

- 오리진 약점: **합성 불가** (단일 능력만 교대) — 주인공과의 결정적 차이
- 드롭: 타워의 심장, 클리어 증표, **해충방제 완료 확인서** ("작업비: 미수금")

## 집필 시 체크리스트

- [ ] 해당 층의 **생태 테마** 확인 (상세본 참조)
- [ ] 등장 몬스터가 해당 구간에 **매핑된 생물**인지
- [ ] 능력 각성이 **매핑된 층**에서 이루어지는지
- [ ] 보스전에 **쾌감 패턴**이 배치되어 있는지
- [ ] 처리 방식에 **해충방제 지식** 연결이 있는지
- [ ] 보스 드롭이 기존 **아이템 설정**과 일치하는지
- [ ] **뒤집기 카운터** 해당 여부 확인
