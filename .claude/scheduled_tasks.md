# Bug Hero 정기 스케줄 목록

세션 시작 시 아래 스케줄을 CronCreate로 등록할 것.
(Claude Code의 CronCreate는 세션 내에서만 유효하며, 7일 후 자동 만료)

## 1. 세션 중간 체크 (매시 :17)
- **Cron**: `17 * * * *`
- **내용**: 현재작업상태.md 최신 여부 + 미커밋 원고 + 중간 저장 권고

## 2. 능력 빈도 감사 (3시간마다 :43)
- **Cron**: `43 */3 * * *`
- **내용**: 최근 5화 내 능력 획득/합성 빈도 규칙 준수 여부

## 3. 진행률 리포트 (4시간마다 :07)
- **Cron**: `7 */4 * * *`
- **내용**: 완료 에피소드 수, 오늘 작업량, 평균 속도
