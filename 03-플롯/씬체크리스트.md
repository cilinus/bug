# Scene Checklist - 씬 체크리스트

현재 작성 중인 씬에 대한 체크리스트. 매 씬 작성 전/후 확인.

## Before
- [ ] 현재작업상태.md 확인
- [ ] 보유 능력 목록 확인 (ability-upgrade-current.md)
- [ ] 파티원 현재 상태 확인
- [ ] 화 성격 유형 선언 (포석/돌파/여운/인간/전환)
- [ ] 씬 클라이맥스 배치 결정 (전면/중간/후면)

## During
- [ ] POV 일관성 (씬 내 전환 없음)
- [ ] 감각 묘사 3감각+
- [ ] 대화 노이즈 포함 (끊김/딴소리/비완성/오해/침묵)
- [ ] 코미디 비트 1개+
- [ ] 능력 사용 시 해당 능력 획득 여부 확인
- [ ] 능력 사용 후 페널티(생체 반동) 묘사
- [ ] 먹치킨 톤 유지 (주인공 여유, 과도한 위기 금지)

## After
- [ ] 분량 확인 (2,000~3,500자)
- [ ] 문장 단편화 검사 (SVO 3연속 금지)
- [ ] 대화 패턴 검사 (분리호칭/말했다 단독 금지)
- [ ] em-dash 5회 이내
- [ ] 어체 일관성 확인
