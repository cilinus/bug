# Main Plot - 메인 플롯

## 전체 구조
- **규모**: 200~280화, 4부작
- **서사 모델**: Tower Progression + Munchkin Rise
- **화당**: 6,000~7,000자, 2~3씬
- **톤**: 밝고 유쾌한 먹치킨 코미디

## 핵심 갈등
- 주인공 vs 타워 (공략 대상이지 위협은 아님)
- 주인공 vs 기득권 (길드/정부/국제 → 가볍게 넘김)
- 파티원 vs 자기 내면 (각자의 과거와 성장)

## Part별 요약

### Part 1: 해충방제사, 타워에 가다 (1~60화)
각성 → 첫 타워 솔로 → 파티 결성 → 중층 돌파 + 첫 합성

### Part 2: 먹치킨의 탄생 (61~140화)
길드 전쟁 → 관리국 압박 → 상층 돌파 → 한국 타워 클리어 (세계 최초)

### Part 3: 세계가 주목하다 (141~220화)
해외 원정 → 니콜라이 대립 → 메인 타워 전반

### Part 4: 해충방제 완료 (221~280화)
메인 타워 후반 → 이세계 세력 → 최종 보스 → 일상 복귀
