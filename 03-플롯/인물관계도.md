# Character Relationship Map - 인물관계도

## 주인공 중심 (초기)

```
                    [강현우] ←견제→ [구태봉] ←관리→ [백서진]
                   (크라운 길드장)     |       (헌터관리국)
                                      |
                    ┌─────────────────┼─────────────────┐
                    |                 |                   |
                [한소라]          [정해민]            [윤기택]
               (탱커/방패)      (딜러/요리사)       (참모/협상)
```

## 파티 관계

| 관계 | 특징 |
|------|------|
| 태봉↔소라 | 신뢰. 소라가 유일하게 태봉을 걱정 |
| 태봉↔해민 | 형제. 먹이고 먹히는 공생 |
| 태봉↔기택 | 사장↔전무. 실무 위임 |
| 소라↔해민 | 수다/츳코미 상대 |
| 소라↔기택 | 세대차이 코미디 |
| 해민↔기택 | 요리↔경제 협업 |
