# 시놉시스 — 벌레 잡던 남자가 타워를 잡다 (가제)

## 1. 작품 개요

| 항목 | 내용 |
|------|------|
| 장르 | 현대 판타지, 타워 공략, 먼치킨, 라이트 코미디 |
| 톤 | 밝고 유쾌. 위기감보다 "이번엔 뭘로 패볼까" 하는 여유 |
| 분량 | 200~300화 (4부 구성) |
| 시점 | 3인칭 제한, 과거형 |
| 씬/화 | 2~3씬 |

### 로그라인

만년 해충방제업자 구태봉(30)은 전 세계에 타워가 출현한 날, 작업 중이던 빌딩 지하에서 정체불명의 생물 코어와 접촉해 "생체모방(Biomimicry)" 능력에 각성한다. 자신의 신체에 자연계 생물들의 초능력을 복제·적용할 수 있는 유일무이한 능력. 다른 헌터들이 피 튀기며 고전하는 타워를 "아, 이거 딱총새우 원리잖아"라며 콧노래 부르며 공략하는, 가장 미천했던 남자의 가장 압도적인 타워 정복기.

### 핵심 쾌감 2축

1. 지식 플렉스 — 다른 헌터가 목숨 걸고 싸우는 몬스터를 "이거 그 원리네" 한마디로 해결
2. 능력 조합 시너지 — 2~3개 능력을 합치면 상상 초월의 상위 능력 탄생. 해충방제 경험이 조합의 열쇠

## 2. 세계관

### 타워 출현

- D-Day: 어느 날 전 세계에 동시에 타워 7개 출현
- 각 타워는 대륙/지역별 하나씩. 한국 타워는 서울 한복판
- 타워 주변에 "이세계 생태계"가 침식 — 몬스터 유출, 식생 변화
- 각국 정부·군대 대응 역부족 → 각성자(헌터) 시대 개막

### 타워 내부 구조

| 요소 | 설명 |
|------|------|
| 층 수 | 타워마다 다름. 한국 타워 = 50층(중급). 메인 타워 = 100층(최상급) |
| 생태계 테마 | 5~10층 단위로 변경 (심해→열대우림→균류동굴→극지→화산 등) |
| 층 보스 | 해당 생태계 최상위 포식자의 변이체. 다른 헌터들은 고전 |
| 안전지대 | 10층마다 휴식·거래 가능한 중간 거점 |
| 타워 코어 | 최상층. 클리어하면 해당 타워의 침식 중단 |

### 사회 변화

| 요소 | 상태 |
|------|------|
| 헌터 길드 | 대형 길드가 타워 공략권 독점. 입장료·아이템 독점으로 막대한 수익 |
| 정부 | 헌터 관리국 설치. 등급제(F~S) 운영. 주인공은 초기 F등급 |
| 일반인 | 타워 주변 피난, 몬스터 유출 피해. 해충방제업 수요 폭락(아이러니) |
| 국제 정세 | 타워 공략 속도 = 국력. 각국이 고급 헌터 영입 경쟁 |
| 분위기 | 어둡지 않음. 뉴스에서 "오늘의 타워 공략 현황", 헌터 랭킹 방송, 굿즈 판매 |

## 3. 주인공

### 구태봉 (30세)

| 항목 | 내용 |
|------|------|
| 직업 | 해충방제업자 (경력 10년). 영세 1인 사업자 |
| 성격 | 느긋하고 실용적. 위기에도 "아 이거 그거네" 하는 여유. 돈 계산 빠르고 생활력 강함 |
| 지식 | 생물학 잡학의 끝판왕. 대학은 안 갔지만 현장 실전 + 독학 덕후 |
| 말투 | 서민적, 직설적. "이거 바퀴 잡을 때랑 같은 원리예요" / "경비 처리 됩니까?" |
| 약점(?) | 굳이 꼽으면 귀찮은 거 싫어함. 정치·서열 싸움 관심 없음 |
| 동기 | 처음엔 "헌터가 돈이 더 되니까". 점점 "이 타워 내가 안 깨면 누가 깨?" |

## 4. 파티원

### ① 한소라 (28세, 여) — 탱커/서포터

| 항목 | 내용 |
|------|------|
| 전직 | 소방관 |
| 각성 능력 | 강화 체질 (체력·방어력 극대화) |
| 성격 | 정의감 강하고 직선적. 주인공의 능력 부작용 볼 때마다 기겁 |
| 파티 역할 | 방패 + 상식인. "그게 진짜 되는 거야?" 담당 |

서사: 소방관 시절 타워 유출 몬스터가 소방서 습격. 동료 3명 사망, 소라만 각성하며 생존. 강화 체질이라 죽지는 않지만 살리지도 못했다는 죄책감. 각성 후 사직, 헌터가 되지만 "강한데 싸우는 법을 모르는" 상태. 대형 길드에서 "탱커는 넘쳐요" 퇴짜.

합류: 타워 저층에서 몬스터에 밀려 고립됐을 때 태봉이 우연히 구출. 근데 구출 방식이 기절염소 능력으로 소라의 근육을 경화시켜 방패 삼아 같이 돌파한 것. "…저를 방패로 쓴 거예요?" / "강화 체질이잖아요. 저보다 단단하신데." 태봉의 능력을 보며 깨달음 — 강한 게 아니라 아는 게 중요하다.

내면 아크: 동료 상실 죄책감 → 태봉 파티에서 "이번엔 지킨다" → "지키는 것"이 아니라 "함께 싸우는 것"

### ② 정해민 (26세, 남) — 딜러

| 항목 | 내용 |
|------|------|
| 전직 | 호텔 셰프 지망생 |
| 각성 능력 | 화염 조작 (흔하디 흔한 불 마법) |
| 성격 | 수다쟁이, 낙천적, 먹는 것에 진심 |
| 파티 역할 | 화력 + 파티 밥 담당. 대사 과부하 페널티 전담 해결사 |

서사: 각성 순간 주방에서 대형 화재. 호텔 전소, 동료 셰프 중상, 해고. "불을 다루는 요리사"가 "불로 모든 걸 태운 남자"가 됨. 화염 능력은 F~D급 중 가장 흔함. 어디서나 "또 불이야?" 화력은 있는데 "제어 못 하면 아군도 태우잖아"로 기피.

합류: 태봉이 안전지대에서 배고파 쓰러져 있을 때, 해민이 몬스터 식재료로 요리해서 먹여줌. "형씨, 나랑 파티 하면 밥값은 확실히 벌어." 태봉의 대사 과부하 = 해민에겐 요리 실력 발휘 무대. "전투 요리사"라는 포지션 발견.

내면 아크: 불로 파괴한 죄책감 → 불이 아닌 요리로 가치 증명 → 화염 제어력 성장 → "불은 태우는 게 아니라 익히는 거야"

### ③ 윤기택 (45세, 남) — 정보/참모

| 항목 | 내용 |
|------|------|
| 전직 | 보험 설계사 |
| 각성 능력 | 확률 직감 (위험도를 0~100 수치로 느끼는 약한 예지) |
| 성격 | 신중하고 계산적. 말수 적지만 핵심을 찌름. 중년의 쓴맛을 아는 사람 |
| 파티 역할 | 두뇌 + 협상. 길드·정부와의 계약, 아이템 매매, 수익 관리 |

서사: 잘 나가던 보험 설계사. 타워 출현 후 보험 시장 붕괴(몬스터 피해 = 면책). 회사 파산, 아내 이혼, 중학생 딸은 아내 쪽. 양육비도 못 주는 처지. 40대 늦깎이 각성, 전투력 제로. "아저씨, 위험한 거 느끼면 뭐 해요? 피하기라도 해요?"

합류: 태봉의 첫 대형 길드 마찰 때, 보험 설계사 시절 인맥+협상력으로 법적 문제 정리. "기택 씨, 이거 한 건에 얼마예요?" / "…공짜입니다. 대신 파티 넣어주세요." 태봉 옆의 위험도가 항상 "0"이거나 "측정 불가"인 게 경이.

내면 아크: "확률적으로 인생은 끝났다" → 태봉이라는 측정 불가의 변수 → "계산 밖의 인생도 있다" → 딸에게 다시 연락하는 용기

## 5. 빌런

### ① 강현우 — 크라운 길드장 (Part 1~2 메인)

| 항목 | 내용 |
|------|------|
| 등급 | S급. 능력: 중력 조작 |
| 위치 | 타워 출현 첫 세대 각성자. 한국 타워 1~20층 최초 공략. 국가 영웅 |
| 동기 | 크라운 길드의 타워 공략 독점권(정부 계약). 태봉이 독점을 무너뜨림 |

척을 진 이유: 크라운 길드가 타워 공략 독점권을 정부와 계약. 입장료·드롭 아이템 독점이 수익원. 태봉이 F등급으로 타워 솔로 입장 → 크라운이 막고 있던 21층을 하루 만에 돌파. 독점 계약 위반. 강현우 입장: "규칙을 지키는 게 질서야. 한 명이 멋대로 하면 전체가 무너져." 반은 맞는 말이지만 본심은 독점 붕괴에 대한 공포.

전개: 견제 → 물리적 충돌 → 패배 → 더 강한 수를 쓰려다 자멸. 최후에 약간의 구원(메인 타워에서 협력).

### ② 백서진 — 헌터관리국 부국장 (Part 2 메인)

| 항목 | 내용 |
|------|------|
| 등급 | 비각성자 |
| 위치 | 행정고시 출신. 헌터 등급제·입장 허가제·드롭 과세 등 제도 설계자 |
| 동기 | 태봉의 능력이 등급 측정 불가. 분류 불가 = 관리 불가 = 위험 요소 |

척을 진 이유: 태봉이 규정 무시, 길드 미가입, 세금 신고도 해충방제업으로 함. "당신이 선의든 악의든, 시스템 밖에 있으면 안 됩니다." 반은 맞는 말이지만 본심은 자신이 만든 시스템의 완전성에 대한 집착.

전개: 회유 → 제도적 압박(면허 정지, 계좌 동결) → 정치적 거래 시도 → 태봉이 너무 신경 안 씀. "국가 안보 사안입니다" / "아 네, 근데 배고파서요"

### ③ 니콜라이 볼코프 — 러시아 최강 헌터 (Part 3 메인)

| 항목 | 내용 |
|------|------|
| 등급 | 세계 최강급. 능력: 빙결 지배 |
| 위치 | 러시아 타워 최상층 도달. UN 국제헌터기구 "인류의 검" |
| 동기 | 메인 타워를 세계 연합팀으로 공략하자는 합의. 태봉이 솔로 진입 거부 |

척을 진 이유: "개인 능력이 뛰어나도 인류의 운명을 혼자 짊어질 순 없다." 틀린 말 아니지만 본심은 "내가 아닌 누군가가 인류를 구한다"는 걸 받아들이기 어려운 자존심.

전개: 경고 → 실력 시험 → 패배 후 인정 → 최종적으로 협력.

### 빌런 설계 원칙

- 전원 "반은 맞는 말"을 함 → 독자도 일리 있다고 느끼지만 주인공 편을 듦
- 강현우(질서 vs 자유 = 기득권), 백서진(시스템 vs 예외 = 통제욕), 니콜라이(집단 vs 개인 = 자존심)
- 셋 다 주인공이 가볍게 넘김. 심각한 갈등이 아니라 "아 또 뭐래" 느낌의 쾌감

## 6. 능력 시스템 — 생체모방(Biomimicry)

### 핵심 개념

주인공의 몸 자체가 생물 능력의 그릇. 스킬 시전이 아니라 신체 변이/강화 방식.

### 규모

| 구분 | 수량 |
|------|------|
| 실제 생물 기반 | ~86종 |
| 창작 능력 | ~10종 |
| 합성 능력 | ~20종+ |
| 총계 | ~116종+ |

### 등급 체계

| 등급 | 비율 | 특성 | 획득 시기 |
|------|------|------|----------|
| 하 (D~C급) | ~35% | 웃기거나 상황 한정적. 주인공의 지식으로 의외의 활용법 발견 | 초반~중반 |
| 중 (B급) | ~40% | 실전에서 유용. 조합 재료로서 가치 큼 | 중반 위주 |
| 상 (A~S급) | ~25% | 단독으로 판을 뒤집는 능력. 합성의 핵심 재료 | 중반~후반 |

### 획득 경로 (이중 트랙)

| 경로 | 방식 | 비율 |
|------|------|------|
| 층 클리어 보상 | 해당 층의 생태계 테마에 맞는 능력 자동 각성 | ~60% |
| 생물 조우 복사 | 타워 내 생물형 몬스터 처치·특수 조우 시 흡수 | ~40% |

### 6-3. 하급 능력의 반전 활용

주인공이 해충방제업자라서 가능한 "쓸모없어 보이는 능력의 천재적 활용". 다른 헌터들이 "그 능력으로 그걸 해?" 하는 지식 플렉스의 핵심.

| 하급 능력 | 다른 헌터 반응 | 주인공 활용 |
|----------|--------------|-----------|
| 웜뱃 (정육면체 똥) | "…뭐?" | 완벽한 정육면체 = 건축 자재. 바리케이드·계단 즉석 생성 |
| 기절염소 (경직) | "방해잖아" | 타격 직전 해당 부위만 경직 → 철골 경도 확보 |
| 매너티 (방귀 부력) | "진지한 거야?" | 수중 완벽 부력 조절. 잠수/부상 자유자재 |
| 나무늘보 (초저대사) | "느려빠져서" | 독·저주 확산 1/10 감속. 해독 시간 확보의 극치 |
| 개복치 (무기력) | "최약체잖아" | 표면적 극대화 → 에너지 흡수 탱크. 발열과 합성 시 인체 열폭탄 |
| 포투새 (나뭇가지 흉내) | "숨는 게 다야?" | 생체 반응(심박·체온·기) 소거 → 모든 탐지 능력 무력화 |
| 회전초 (뿌리 절단) | "잡초잖아" | 에너지 연결(버프/저주/기생)을 물리 절단. 디스펠 |
| 코알라 (저지능) | "…이게 능력?" | 정신 공격 면역. 마인드컨트롤·환각 시 "생각 자체를 안 함" |

| 카카포 (추락 앵무) | "바보새잖아" | 고소 공포 완전 제거. 송골매 급강하와 찰떡 시너지 |
| 호아친 (소똥 냄새) | "냄새 무기?" | 극악취로 접근 차단. 밀폐 공간에서 범위 무력화 |

### 6-4. 페널티 시스템 — 생체 반동(Biofeedback)

자연계 생물은 능력을 위해 무언가를 포기했다. 주인공도 마찬가지.
단, 페널티는 위기가 아니라 개그와 전략적 선택지로 작동.

#### 티어별 반동

| 등급 | 반동 수준 | 톤 |
|------|----------|-----|
| 하 | 거의 없음. 있어도 웃긴 수준 | 코미디 |
| 중 | 불편하지만 치명적이지 않음 | 약간의 긴장 |
| 상 | 확실한 대가. 전략적 판단 필요 | 쾌감 있는 선택 |
| 합성(SS) | 큰 대가. 주인공은 지식으로 줄임 | "그걸 그렇게 피해?" |

#### 반동 유형 5가지

(1) 대사 과부하 — "배고파 죽겠다"
- 강한 능력일수록 칼로리 소모 폭증
- 맨티스슈림프 펀치 한 방 = 하루치 칼로리. 합성 쓰면 쓰러질 만큼 허기
- 타워 공략 중 간식 잔뜩 챙기는 러닝 개그
- "형, 또 먹어요?" / "이거 경비 처리 되는 거지?"

(2) 신체 역변이 — "잠깐 좀 이상해진다"
- 능력 사용 후 해당 생물 특징이 외형에 일시 잔류
- 딱총새우 쓰면 손가락 한쪽 비대(30분), 갑오징어 풀면 피부 얼룩(1시간)
- 상급일수록 오래, 눈에 띄게 → "괴물 아니야?" 소동 = 코미디

(3) 감각 잔상 — "아직 그 감각이 남아있어"
- 감각 능력 해제 후 해당 감각 과민 잔류
- 박쥐 끄면 소음 민감(두통), 살무사 끄면 따뜻한 게 다 빛남
- "형광등 좀 꺼줘… 머리 깨질 것 같아"

(4) 생물학적 회복기 — 쿨타임이 아닌 재충전
- 전기세포 재충전에 영양분+시간, 독선 재생산에 특정 음식 섭취
- 주인공이 해충방제 지식으로 회복 단축 = 쾌감 ("이 성분이면 독 재합성 3배 빨라")

(5) 동시 사용 제한 — "몸은 하나"
- 기본 능력 동시 3개 자유, 4~5개 부담, 6개+ 한계
- 합성은 2~3슬롯 차지
- 상충 능력 동시 사용 불가 (갑오징어 위장=정지 + 송골매 급강하=운동)
- "이 상황에서 어떤 조합?" → 최적 조합을 바로 찾음 = 먼치킨 유지

#### 성장에 따른 페널티 변화

- 초반: 상급 하나 쓰면 반나절 역변이. 합성은 시도 자체가 모험
- 중반: 몸이 적응하며 반동 절반. 합성 가능해짐
- 후반: 대부분 반동 경미. SS급 합성만 유의미한 대가
- 먼치킨 톤 유지: 어느 시점이든 반동 때문에 위기에 빠지지는 않음

## 7. 플롯 구조 — 4부 (200~280화)

### Part 1: 해충방제사, 타워에 가다 (1~60화)

| 아크 | 화수 | 핵심 |
|------|------|------|
| 각성 아크 | 1~10 | 타워 출현, 해충방제 중 생물 코어 접촉, 생체모방 각성. F등급 판정 |
| 첫 타워 입장 | 11~25 | 한국 타워 저층(1~10층) 솔로 공략. 하급 능력으로 쉽게 돌파 |
| 파티 결성 | 26~40 | 소라·해민·기택 합류. 각자 사연. 크라운 길드 첫 마찰 |
| 중층 돌파 | 41~60 | 11~20층. 첫 합성(멸살권). 강현우 첫 대면. 백서진 등장 |

톤: 가장 가볍고 유쾌. "바퀴 잡을 때랑 같은 원리예요"
핵심 쾌감: 무시→반전 반복. F등급이 S급 길드 막힌 층을 뚫는 사이다
능력: 하~중급 20종. 단일 능력으로도 압도

### Part 2: 먹치킨의 탄생 (61~140화)

| 아크 | 화수 | 핵심 |
|------|------|------|
| 길드 전쟁 | 61~85 | 크라운 견제, PK, 여론전. 강현우 직접 대결 → 패배 |
| 관리국 압박 | 86~105 | 백서진 제도적 공격. 기택 협상 활약 |
| 상층 돌파 | 106~125 | 21~40층. 합성 본격화. 파티원 성장 에피소드 |
| 한국 타워 클리어 | 126~140 | 41~50층+최종 보스. 세계 최초 클리어 |

톤: 유쾌+스케일 확대. 갈등 있지만 심각하지 않음
핵심 쾌감: 합성 조합의 창의적 활용. "그 둘을 합치면 그게 돼?"
능력: 중~상급 30종. 합성 본격화
파티원 서사: 소라(동료 극복), 해민(화염 제어), 기택(딸 재연결)

### Part 3: 세계가 주목하다 (141~220화)

| 아크 | 화수 | 핵심 |
|------|------|------|
| 해외 원정 | 141~165 | 타국 타워 지원. 해외 랭커 조우. "한국의 해충업자" 별명 |
| 니콜라이 대립 | 166~190 | 메인 타워 출현. 연합팀 vs 솔로. 니콜라이 대결 → 인정 |
| 메인 타워 전반 | 191~220 | 메인 타워(100층) 진입. 연합팀과 비공식 협력 |

톤: 스케일 커지지만 톤 유지. 해외에서도 서민적 태도
핵심 쾌감: 세계 최강 헌터들 경악. 국제 무대 지식 플렉스
능력: 상급 20종. SS급 합성 등장

### Part 4: 해충방제 완료 (221~280화)

| 아크 | 화수 | 핵심 |
|------|------|------|
| 메인 타워 후반 | 221~250 | 51~90층. 이세계 생태계. 창작 능력 획득 |
| 이세계 세력 대면 | 251~265 | 타워를 보낸 존재 조우. 타워 = "지구 생태계 테스트". "해충 검사였다고요?" |
| 최종 보스 | 266~275 | 100층. 전 능력+전 합성 총동원. 살짝 고생(그래도 이김) |
| 에필로그 | 276~280 | 타워 소멸. 세계 복구. 해충방제업 재개. "타워도 큰 해충 둥지였지, 뭐" |

톤: 스케일 최대. 결말은 일상 회귀
핵심 쾌감: 총집결. 전 능력+전 합성 퍼레이드
반전: 타워가 침략이 아니라 "테스트"였다는 가벼운 반전
최종 메시지: 대단한 능력보다 그걸 이해하고 활용하는 지식과 경험이 진짜 힘

### 서브플롯 흐름

| 서브플롯 | 전개 |
|---------|------|
| 태봉의 사업 | 해충방제업 → 헌터 겸업 → "몬스터 방제 전문" 법인 → 세계적 기업(본인 무관심, 기택 운영) |
| 소라의 치유 | 동료 상실 → "이번엔 지킨다" → "지키는 것"이 아니라 "함께 싸우는 것" |
| 해민의 불꽃 | 파괴의 불 → 요리의 불 → 전투 요리사 → "불은 태우는 게 아니라 익히는 거야" |
| 기택의 재기 | 인생 끝났다 → 측정 불가의 변수 → 계산 밖의 인생 → 딸에게 연락하는 용기 |
| 강현우 | 영웅 → 기득권 → 패배 → 자멸 → 최후에 메인 타워 협력(약간의 구원) |
