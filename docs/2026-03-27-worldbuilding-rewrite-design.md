# 세계관 리라이트 설계서

**날짜**: 2026-03-27
**범위**: 세계관 설정 변경 + 전체 원고(1~9화) 리라이트/수정

---

## 1. 핵심 변경: 타워 5년+ 경과 세계

### 타워
- D-Day는 5년+ 전. 타워는 이미 일상의 일부
- 전 세계 7개 타워, 각국 공략 경쟁 진행 중
- 한국 타워 서울 소재, 현재 공략 진도 존재

### 마나
- 고착화된 상식적 개념
- 타워 주변 마나 농도 높음 → 변이충 생성 원인
- 각성자의 능력 원천

### 각성 유형
| 유형 | 설명 |
|------|------|
| 자연각성 | 로또 수준 희귀. 무작위 발현 |
| 강제각성 | 각성열매 섭취. 일반인은 꿈도 못 꿀 만큼 비쌈 |
| 태봉(유니크) | 트루스 촉수 접촉. 능력 종합 전이 |

### 각성열매
- 타워 내 지성 변이체 **트루스(Truce)**가 촉수로 특정 열매에 능력을 삽입하여 생성
- 타워 드롭 아이템 중 최고가
- 길드가 독점적으로 수급

### 트루스(Truce)
- 타워 내부의 지성 변이체
- 촉수를 통해 능력을 열매에 삽입하는 고유 능력 보유
- 연구를 위해 생포되어 타워 외부 연구시설에서 연구 중이었음
- 작중 탈옥 → 태봉과 조우 → 능력 상실 후 회수

### 등급제 (S~F, 5년째 운영)
| 등급 | 비율 | 설명 |
|------|------|------|
| S | 1%↓ | 최상위. 국가 영웅급. 연봉 수백억 |
| A | 3% | 국가급 전력. 연봉 수십억 |
| B | 5% | 상층 가능. 길드 간부급 |
| C | 10% | 중층 핵심 전력 |
| D | 15% | 단독 저층, 중층 파티원 |
| E | 25% | 소규모 파티 저층 가능 |
| F | 40% | 저층 몬스터 상대 |
| 측정불가 | 극소수 | 기존 체계 분류 불가. **구태봉** |

### 사회구조
- 각성자 = 상류 계급. 높은 수입, 사회적 존경
- 비각성자는 위험하지만 각성자가 기피하는 하층 일 담당
- 변이충 방제 = 대표적 하층 위험직 (각성자들은 "하찮은 일"로 무시)
- 계급 고착화 심화 중

## 2. 주인공 캐릭터 추가 설정

### 구태봉 — 박찬호형 수다
- **하층민의 삶을 견디기 위해 말이 많아짐**
- 박찬호 밈 참조: TMI 폭격, 본론 없이 샛길로 빠짐, 긴장하면 더 심해짐
- 혼자 일하는 시간이 길어서 벌레한테 말 거는 버릇 + 사람 만나면 수다 폭발
- 개그 엔진으로 활용: 진지한 상황에서 갑자기 방제 경험담, 돈 계산 TMI, 벌레 생태 강의
- 서술(내면 독백)에서도 이 성격 반영: 생각이 꼬리에 꼬리를 물고 삼천포로

### 각성 증상 — 보라색 눈동자
- 일반 각성자: 각성 시 한쪽 눈동자가 은색으로 영구 변화
- 태봉(유니크): 능력 발현 시에만 한쪽 눈동자가 보라색으로 변화, 평소에는 정상
- 이 특성 때문에 평소에는 각성자인지 알 수 없음

## 3. 에피소드 구조 (1~9화)

### 1화 — 해충방제사의 하루 (세계관 확립)
- 타워가 있는 세상에서 태봉의 변이충 방제 일상
- 위험한 하층 직업임을 보여주기: 변이충의 위험성, 각성자들의 무시
- 태봉 캐릭터 확립: 돈 계산, 방제 지식 플렉스, 벌레한테 말걸기, 수다
- 세계관 자연 노출: 등급, 길드, 마나, 각성열매 등 (설명 아닌 일상 속에서)

### 2화 — 복선 + 일상 심화
- 방제 일상 계속 + 변이충이 최근 더 커지고 있다는 흐름
- 뉴스: "타워 연구시설 보안 사고" (트루스 탈옥 복선, 트리거 이벤트 무게감 부여)
- 태봉의 캐릭터 심화 (주변인과의 대화, 돈 걱정, 하층민의 현실)
- 기존 1화에서 확립한 톤 유지

### 3화 — 트루스 조우 (인싸이팅 이벤트)
- 지하실 방제 작업 중 약해진 트루스와 조우
- 트루스는 연구실에서 약해질 대로 약해진 상태 (원래라면 태봉 즉사급)
- 이판사판으로 맹독(방제 약제) 주입
- 트루스가 능력전이로 맹독을 태봉에게 반환 시도
- 태봉이 맹독을 다 돌려받은 이후에도 촉수를 놓지 않음
- → 능력 종합 전이 발생, 트루스 능력 상실
- 군/연구팀 도착, 트루스 회수. 태봉은 멍한 상태로 귀가

### 4화 — 능력 발견
- 방제 중 벌레 감지 능력 발동
- 동료/의뢰인이 "형, 눈이..." → 보라색 눈동자 발견
- 벤조퀴논 첫 발동 (생물학 지식 유입)
- 각성 인지 → 등록해야 하나 고민

### 5화 — 능력 실험 + 영업 확장
- 능력 실험, 페널티 발견 (배고픔, 손 빨개짐)
- 변이충 방제 효율 폭증 → 수입 급증
- 기존 방제 기술 + 능력 시너지

### 6화 — 영역 확장
- 의뢰 폭증, 먼 지역까지 출장
- 방제업계 소문 확산: "변이충 잘 잡는 방제업자"
- 박진수 만남

### 7화 — 지네전 + 딱총새우
- 지네 변이충 조우, 벤조퀴논 상성 안 맞음
- 딱총새우 충격파 첫 발동 (위기 상황)

### 8화 — 딱총새우 연습
- 원룸 실험, 성공률 향상
- 진수 전화, 지네 확산 정보
- 각성자 등록에 대한 고민

### 9화 — F등급 등록 + 크라운 길드
- 능력 사용 시 눈동자 변화 → 들킬 위험 때문에 등록 결심
- 이미 확립된 관리국에서 등록, 측정 불가 → F등급
- 해외 S급 사례 뉴스 (백억 계약 등)
- 크라운 길드 접촉

## 5. 주의사항

- 기존 원고의 **톤, 캐릭터 보이스, 방제 지식, 돈 계산 유머**는 최대한 보존
- 기존 원고에서 살릴 수 있는 비트/대화는 살릴 것
- core-rules.md의 모든 규칙 준수 (특히 §4 문장 단편화 금지, §4-1 웹소설 톤)
- 박찬호형 수다 = 새 개그 엔진, 모든 화에 자연스럽게 반영
- 트루스 탈옥 뉴스(2화)는 §3-3 트리거 무게감 규칙 적용
