# 대화 위계 규칙서 — Bug Hero

등장인물 간 호칭/말투/격식의 변화를 정리한 문서.

## 파티 내부

| 화자 → 대상 | 호칭 | 어체 | 변화 시점 |
|------------|------|------|----------|
| 태봉 → 소라 | 소라 씨 → 소라 | 해요체 → 반말 | Part 2 중반 |
| 태봉 → 해민 | 해민 씨 → 해민아 | 해요체 → 반말 | Part 1 후반 |
| 태봉 → 기택 | 기택 씨 | 해요체 (유지) | - |
| 소라 → 태봉 | 태봉 씨 → 오빠/태봉 | 해요체 → 반말 | Part 2 후반 |
| 해민 → 태봉 | 형 | 반말 (처음부터) | - |
| 기택 → 태봉 | 구 대표 | 합쇼체 (유지) | - |
| 소라 ↔ 해민 | 소라 씨/해민 씨 → 이름 | 해요체 → 반말 | Part 1 후반 |
| 소라 ↔ 기택 | 기택 씨/소라 씨 | 해요체 (유지) | - |
| 해민 ↔ 기택 | 아저씨/해민 군 | 반말/합쇼체 | - |

## 외부 인물

| 관계 | 호칭 | 어체 |
|------|------|------|
| 태봉 → 강현우 | 강 길드장님 | 해요체 (무관심 톤) |
| 태봉 → 백서진 | 부국장님 | 해요체 |
| 강현우 → 태봉 | 구태봉 씨 → 구태봉 | 존대 → 반말 (적대 심화) |
| 백서진 → 태봉 | 구태봉 씨 | 합쇼체 (관료적) |
| 니콜라이 → 태봉 | Mr. Gu / 구 씨 | 존대 (영어 혼용) |

## 원칙
- 한 발화 내 존대/반말 혼용 절대 금지
- 전환 시점은 명확한 사건 계기 필요
- 캐릭터 음량: 기택(짧고 핵심만), 해민(길고 수다), 소라(중간, 직설), 태봉(상황따라 유연)
