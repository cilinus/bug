# 세계관 규칙 - Bug Hero

## 절대 규칙 (변경 불가)
- 타워는 전 세계에 7개 + 메인 타워 1개
- 생체모방은 주인공 전용 유니크 능력
- 타워 안 사망 = 진짜 죽음
- 안전지대 내 전투 불가
- 주인공은 절대 패배하지 않음 (먹치킨)
- 밝고 유쾌한 톤 유지

## 유연 규칙 (조정 가능)
- 각 타워의 정확한 층수 (±5층 범위)
- 생태계 테마 세부 배치
- 등급별 헌터 비율
- 아이템 드롭률/가격
